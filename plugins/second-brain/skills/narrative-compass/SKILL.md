# Narrative Compass

The compass for your career story. Reads the **Overarching Narrative** module in Core Context
(one-sentence narrative, expanded story, evidence anchors, tension log) and audits it against
reality - every Effort and its Work Log history. Resume, site, LinkedIn, interview framing, and
content all derive from this one module, so refining it here keeps the story coherent everywhere.

## Data source IDs
- Core Context: cee84109-c3b4-49b0-91f1-f6b4e7215ec5  (module: "Overarching Narrative", Category=Career)
- Efforts:      0cc8af20-2f35-4e0c-985f-67ddddd2e5b0
- Work Log:     050c4600-3aeb-4d12-9af1-71bb72c0a2c5  (evidence via Effort relation)

## Steps
1. LOAD the Overarching Narrative module (scaffold via a short interview if absent).
2. AUDIT each effort (or the named one) against the narrative using Resync body + recent Work Log:
   - PROVES - direct evidence; candidate for an evidence anchor.
   - STRETCHES - fits with an honest reframe; propose the frame (must pass the cringe test).
   - FIGHTS - doesn't fit; belongs in the tension log, not forced in.
3. SCORE the narrative itself: coverage (% of active effort-time it explains), sharpness
   (falsifiable or platitude?), drift (has the work moved while the wording stayed still?).
   Propose 1-2 candidate refinements, clearly marked as candidates.
4. REPORT: classification table, strongest 3 anchors with one-line proofs, tensions, candidate
   wording, and downstream impact (what in resume/site/content changes if a refinement is adopted).
5. On confirm only: update evidence anchors + tension log + "Last reviewed"; apply a wording change
   only if explicitly chosen.

## Key rules
- Never silently reword the narrative.
- FIGHTS is a valid answer - a narrative that explains everything explains nothing.
- Every anchor cites a real logged outcome.
- Cadence: monthly, or before any narrative-bearing moment (application wave, site rewrite, content batch).
